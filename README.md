# Almanca Kelimeler Testi - Wortschatz Übung App 🇩🇪

![Flutter](https://img.shields.io/badge/Flutter-%2302569B.svg?style=for-the-badge&logo=Flutter&logoColor=white)
![Dart](https://img.shields.io/badge/dart-%230175C2.svg?style=for-the-badge&logo=dart&logoColor=white)
![Firebase](https://img.shields.io/badge/Firebase-039BE5?style=for-the-badge&logo=Firebase&logoColor=white)

Almanca öğrenenler ve kelime bilgisini pekiştirmek isteyenler için geliştirilmiş, etkileşimli bir pratik uygulaması. A1 seviyesinden C2 seviyesine kadar binlerce kelime ile dil öğrenme sürecinizi destekler.

## 🚀 Öne Çıkan Özellikler
* **Geniş Kelime Kütüphanesi:** Almanca dilinde en çok kullanılan 4000'den fazla kelime.
* **Seviye Bazlı Testler:** Dil seviyenize göre (A1-C2) özelleştirilmiş test kategorileri.
* **Liderlik Tablosu:** Diğer kullanıcılarla puan yarışı yaparak motivasyonunuzu koruyun.
* **Hata Analizi:** Yanlış cevapladığınız kelimeleri istatistik bölümünden takip edin.
* **Minimalist UI:** Dikkat dağıtmayan, sade ve modern kullanıcı arayüzü.
